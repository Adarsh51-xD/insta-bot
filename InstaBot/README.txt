To run the bot:

1. Get the access_token from the instagram developer: Refer to ACCESS_TOKEN_GENERATION.txt

2. Get the postID from the post link - https://www.instagram.com/p/{post-shortcode}/. Here post-shortcode is the postID

3. Install dependencies:
	a. Node should be installed
	b. Open cmd in the script directory and run - "npm install node-fetch"

4. Run the script:
	node like_bot.js
